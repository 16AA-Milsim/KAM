---------
+ [Kellerkompanie](http://kellerkompanie.com/)

## Core features
--------

- aceAirway
  - [x] Airwaymanagement for unconscious patients
    - [x] Head overstretching
    - [x] Intubation
    - [x] Aspiration
- aceBreathing
  - [x] Oxygen saturation
- aceCirculation
  - [x] Blood Groups
  - [x] Painkillers
  - [x] Bigger AED with monitoring
- aceDisability
  - [x] Cut of clothes
  - [x] Allergical reactions
- aceExposure
  - [ ] Heat-preserving
- aceMisc
  - [ ] Big Backpack?
  - [x] Workable IV stand
